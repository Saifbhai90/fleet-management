Email backup ke liye kya chahiye?

Backup email bhejne ke liye ye environment variables set karein:

  • MAIL_SERVER – SMTP server (e.g. smtp.gmail.com)
  • MAIL_PORT – Port (587 TLS ke liye, 465 SSL)
  • MAIL_USERNAME – Your email / SMTP user
  • MAIL_PASSWORD – Password ya App Password (Gmail ke liye Google Account → Security → App passwords use karein)
  • MAIL_FROM (optional) – Sender email agar alag chahiye
  • MAIL_USE_TLS (optional) – true for port 587

Auto (scheduled) email: BACKUP_EMAIL_TO=receive@email.com set karein; scheduled backup us email par backup bhej dega.

Configuration
  • Download – koi config nahi chahiye.
  • Email – .env mein MAIL_SERVER, MAIL_PORT, MAIL_USERNAME, MAIL_PASSWORD, MAIL_FROM, MAIL_USE_TLS set karein.
  • Save to path – backup lete waqt path khud type karein (optional: default path ke liye BACKUP_PATH .env mein set karein).
