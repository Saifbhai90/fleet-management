Render / Linux deploy

  1. Web service par env add karein: BOOTSTRAP_NODE_FOR_BUILD=1 (taake build waqt Node mil jaye).
  2. Build Command ke end mein yeh lagayein (pip ke baad):
    chmod +x scripts/build_fleet_personal_pc.sh && ./scripts/build_fleet_personal_pc.sh
  3. Deploy dubara chalayein — deploy ke baad static/fleet_personal_pc/index.html image par aa jana chahiye.
