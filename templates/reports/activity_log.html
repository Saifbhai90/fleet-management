Latitude, Longitude aur View on Map kyun kabhi-kabhi nahi dikhte?
Pehli table (Login sessions): Ye server-side hai — sirf IP address log hota hai. Is table mein lat/long hoti hi nahi.
Dusri table (Client activity): Yahan lat/long tab aati hai jab user browser mein location allow karta hai (login/dashboard ke baad ek baar popup aata hai). Agar user ne location block kar di ya browser ne coordinates nahi diye (e.g. indoor, ya permission deny), to entries to banaenge lekin Latitude/Longitude aur View on Map - dikhenge. Device ID aur IP phir bhi save hote hain.
Location popup na aaye ya "Allow" greyed out ho? Browser location sirf HTTPS ya localhost (127.0.0.1) par allow karta hai. Agar app http://192.168.18.15:5000 jaisi HTTP URL se chal rahi hai, to browser location ko block kar deta hai — na popup aata hai, na site settings mein Allow enable hota. Fix: App ko HTTPS par chalao (e.g. production par) ya local test ke liye http://127.0.0.1:5000 use karo.
