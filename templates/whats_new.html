What's New in Fleet Manager

This page shows recent updates in the software so users can quickly see what changed.
Latest Realtime search + Updates bell + AI reports
Recent updates
  • Navbar notification bell now shows unread count and works across all pages.
  • Realtime search on list pages: Drivers, Vehicles, Projects, Companies, Districts, Parking, Party, Products.
  • AI reports can now show fuel expenses, maintenance expenses, penalties, and attendance summary.
  • New dashboard notification: alerts when many active drivers have no attendance in the last 7 days.
Project & Forms improvements
  • Project Entry Form date validation fixed and errors are shown clearly.
  • District form now shows a friendly message if a duplicate district name is entered.
Lists & Exports
  • Drivers, Vehicles, and Projects lists now have CSV export buttons.
  • Export respects current search filters.

For future updates, admins can edit this page (`whats_new.html`) to add new items in the timeline.
